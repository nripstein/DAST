with AGHQ, directly using the AGHQ model's summary information.

``` r

summary(object, ...)
```

## Arguments

- object:

  A 'disag_model_mmap_aghq' object.

- ...:

  Additional arguments (not used).

## Value

An object of class 'summary.disag_model_mmap_aghq' containing the
summary information.
